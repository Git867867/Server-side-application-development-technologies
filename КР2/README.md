# FastAPI Контрольная работа №2

## Описание
FastAPI приложение для контрольной работы по дисциплине "Технологии разработки серверных приложений".

## Выполненные задания
- Задание 3.1: POST /create_user - создание пользователя с валидацией
- Задание 3.2: GET /product/{id} и GET /products/search - работа с продуктами
- Задание 5.1: POST /login - аутентификация с установкой cookie
- Задание 5.2: Подпись session_token с использованием itsdangerous
- Задание 5.3: Динамическое продление сессии (3-5 минут)
- Задание 5.4: GET /headers и GET /info - работа с заголовками
